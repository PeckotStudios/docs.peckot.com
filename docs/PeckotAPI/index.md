# 项目简介

## 服务介绍

PeckotAPI 是一个公共的网络接口服务平台，它提供了多个接口服务以满足不同需求。我们相信这些接口可以帮助用户更快地获取所需要的信息，无论是个人用户还是企业用户。

:::caution
由于该服务使用 [Vercel](https://vercel.com/home) 免费提供的 Serverless Function 服务，故部分不稳定或未开放的接口可能仍能在当前页面寻找到相关信息。
:::
